# How to integrate Google SEO and Google Analytics for Next.js App Router

## 1. Google Analytics (GA4) Integration

Next.js provides an official package that makes integrating Google Analytics simple and optimized.

**Step 1:** Install the `@next/third-parties` package in your `frontend/ems-webapp` directory:
```bash
cd frontend/ems-webapp
npm install @next/third-parties
```

**Step 2:** Add the `GoogleAnalytics` component to your root layout (`frontend/ems-webapp/src/app/layout.tsx`):
```tsx
import type { Metadata } from "next";
import { Inter } from "next/font/google";
import "./globals.css";
import AuthProvider from "@/lib/AuthContext";
import { ProfileProvider } from "@/lib/ProfileContext";
import { GoogleAnalytics } from "@next/third-parties/google"; // 1. Import the component

// ...

export default function RootLayout({ children }: Readonly<{ children: React.ReactNode }>) {
  return (
    <html lang="en" className={`${inter.variable} h-full antialiased`} suppressHydrationWarning>
      <body className="..." suppressHydrationWarning>
        <AuthProvider>
          <ProfileProvider>
            {children}
          </ProfileProvider>
        </AuthProvider>
        
        {/* 2. Add your Measurement ID here */}
        <GoogleAnalytics gaId="G-XYZ1234567" /> 
      </body>
    </html>
  );
}
```

## 2. Google SEO Integration

Next.js App Router has powerful built-in SEO features. Focus on three main areas:

**A. Enhanced Metadata in `layout.tsx` (and `page.tsx` for specific pages):**
Expand the basic metadata in `layout.tsx` to include Open Graph (social sharing) tags, keywords, and canonical URLs:
```tsx
export const metadata: Metadata = {
  title: {
    template: '%s | EMS',
    default: 'EMS | Precision In Power',
  },
  description: "Advanced energy management system.",
  keywords: ["energy management", "IoT", "power analytics"],
  openGraph: {
    title: "EMS | Precision In Power",
    description: "Advanced energy management system.",
    url: "https://your-production-domain.com",
    siteName: "EMS",
    type: "website",
  },
  robots: {
    index: true,
    follow: true,
  }
};
```

**B. Generate a Sitemap (`src/app/sitemap.ts`):**
Create a `sitemap.ts` file in your `app` directory. Next.js will automatically generate a `sitemap.xml` for Google to index your site efficiently.
```typescript
import { MetadataRoute } from 'next'
 
export default function sitemap(): MetadataRoute.Sitemap {
  return [
    {
      url: 'https://your-production-domain.com',
      lastModified: new Date(),
      changeFrequency: 'yearly',
      priority: 1,
    },
    // Add other public routes like /login, /signup here...
  ]
}
```

**C. Generate a `robots.txt` (`src/app/robots.ts`):**
Create a `robots.ts` file in your `app` directory to guide search engine crawlers on what they should or shouldn't index (e.g., blocking the `/dashboard` routes).
```typescript
import { MetadataRoute } from 'next'
 
export default function robots(): MetadataRoute.Robots {
  return {
    rules: {
      userAgent: '*',
      allow: '/',
      disallow: ['/dashboard/', '/analytics/', '/settings/'], // Protect private routes
    },
    sitemap: 'https://your-production-domain.com/sitemap.xml',
  }
}
```
