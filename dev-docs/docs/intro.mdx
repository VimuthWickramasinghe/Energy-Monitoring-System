# Energy Monitoring System (EMS)

Welcome to the **Energy Monitoring System** documentation. This system provides real-time insights into energy consumption, efficiency metrics, and hardware status.

## Project Overview

The EMS is designed to aggregate data from smart meters and IoT sensors to help users optimize their energy usage and reduce costs.

### Key Features

- **Real-time Dashboard**: Live visualization of power consumption.
- **Historical Analysis**: Track energy trends over days, months, or years.
- **Alerting System**: Notifications for unusual spikes or hardware failures.
- **Exportable Reports**: Generate PDF/CSV summaries for auditing.

## Tech Stack

- **Dev_docs**: React with Docusaurus for documentation.
- **frontend**: Next js.
- **Backend**: Node.js / Express.
- **Hardware**: ESP32 and SCT current clamp and ZMP voltage transformer .
- **Database**: mongoDB and supabase.

## Quick Start for Developers

To run the documentation locally, navigate to the `dev-docs` directory and run `npm install` followed by `npm start`.
The documentation will be available at `http://localhost:3000`.
